# demo

_A demo CLI to showcase swift-cli-docs-plugin._

**Usage**

```bash
demo <subcommand>
```

This tool exists only so you can run `swift package generate-docs` and see
what kind of Markdown the plugin produces for a small Argument Parser CLI.

## Subcommands

| Command | Description |
| --- | --- |
| [`build`](demo-build.md) | Build the demo project. |
| [`test`](demo-test.md) | Run the test suite. |
| [`help`](demo-help.md) | Show subcommand help information. |
