# demo test

_Run the test suite._

**Usage**

```bash
demo test [--filter <filter>] [--verbose]
```

## Options

| Name | Default | Description |
| --- | --- | --- |
| `--filter <filter>` | `—` | Filter tests by name. |

## Flags

| Name | Default | Description |
| --- | --- | --- |
| `--verbose` | `—` | Show verbose output. |
